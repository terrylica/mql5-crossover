# ZigZag Indicator Suite

## Files

| File | Purpose |
|------|---------|
| `ZigZag_Color_NoRepaint.mq5` | Production indicator for MT5 charts |
| `ZigZag_Python_Reference.mq5` | Same indicator with extensive Python porting documentation |

## Usage

- **For MT5**: Use `ZigZag_Color_NoRepaint.mq5`
- **For Python migration**: Reference `ZigZag_Python_Reference.mq5` for algorithm documentation

## Features

- Color-coded zigzag lines (up/down trends)
- Confirmation arrows at reversal points
- No-repaint mode (optional)
- Customizable depth, deviation, and backstep parameters
